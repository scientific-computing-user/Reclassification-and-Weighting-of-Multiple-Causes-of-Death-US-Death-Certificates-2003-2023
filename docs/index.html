What This Project Shows

This project studies how U.S. death-certificate causes changed after expert recoding (Entity Axis → Record Axis) and how weighting multiple listed causes changes the apparent mortality burden across disease groups (2003–2023).

Deaths analyzed
56,986,831
Years covered
2003–2023
Main signal
Cause shifts are substantial
Reproducibility
Command-level, script-based

Plain-Language Findings

  • Recoding changes which condition is counted as the underlying cause for a meaningful fraction of deaths.
  • COVID-19 underlying counts increase strongly after recoding; some other categories move in the opposite direction.
  • When multiple causes are weighted (instead of attributing everything to one cause), burden shifts notably across categories.
  • Trend signals differ by age group; some categories rise over time while others decline in older populations.

All of the above is backed by reproducible scripts in this repository. Outputs are generated into results/reproduced/.

How To Reproduce

  • Install dependencies: python3 -m pip install -r requirements.txt
  • Run the packaged workflow: bash scripts/reproducibility/reproduce_snapshot.sh
  • The workflow auto-rebuilds all_I=converted_with_original.csv from split chunks in data/processed_snapshot/all_I_parts/.
  • Run claim validation directly: python3 scripts/reproducibility/validate_claims.py --all-i data/processed_snapshot/all_I=converted_with_original.csv --lookup data/reference/icd10_to_DL_lookup_v4.csv --entity-column sel_code --output-json results/validation/claim_metrics.json --output-md results/validation/claim_check.md

The packaged run writes outputs to results/reproduced/ and rebuilds required large-source inputs locally from chunked files.

Transparency Notes

  • The bundled snapshot reproduces most headline values closely but not every transition claim exactly.
  • Differences are surfaced explicitly in the automated claim-check report rather than hidden.
  • The report includes both observed values and paper targets for direct inspection.

This is a research reproducibility package, not clinical guidance.
