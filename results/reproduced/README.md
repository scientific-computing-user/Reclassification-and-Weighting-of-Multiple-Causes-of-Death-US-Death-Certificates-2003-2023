# `results/reproduced/`

Reproduction outputs generated by `scripts/reproducibility/reproduce_snapshot.sh`.

## Subfolders

- `tables/` — spreadsheet tables.
- `figures/` — figure image outputs.
- `supplement/` — supplemental derived files.
- `validation/` — run-level claim-check copies from snapshot reruns.
- `logs/` — step-by-step execution logs.
